Beginner:

It is the default level associated to new users. Beginner users have limited access to the platform. They can execute up to 1 concurrently simulation and they can not perform any write operation on the group folders (i.e. they have read access only).

 

Advanced:

Advanced users have unlimited access to the platform. To become an advanced user you should:

  1. Request a grid certificate from your local Certification Authority
  2. Register in the Biomed Virtual Organization
  3. Contact the platform administrators
